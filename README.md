Система автоматизации для записи на занятие в фитнес-центр. Проект был написан для личного пользования.
Позволяет не отслеживать и ждать время записи, а делать это автоматически, выбрав нужные занятия заранее.
- Выбор нужного занятия через ТГ-бот
- Создание крон задачи для записи в определенное время
- Осуществление записи на занятие + отправка уведомления пользователю
